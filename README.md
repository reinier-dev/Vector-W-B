# Vector W&B: Your Digital Flight Partner

## Overview
A modern and intuitive tool for pilots and aviation enthusiasts. This demo app streamlines essential aircraft weight and balance calculations, featuring a dynamic CG envelope graph, unit conversions, and aircraft profile management.

## Features
- **Interactive CG Envelope:** Visualize your aircraft's center of gravity in real-time on a graphical envelope.
- **Dynamic Calculations:** Instantly see changes to weight, moment, and CG as you update passenger, fuel, and cargo data.
- **Unit Conversion:** Seamlessly switch between common aviation units (lbs, kgs, etc.).
- **Aircraft Profile Management:** Save and load different aircraft specifications and weight limits.

## Technologies Used
- **Frontend:** React, JavaScript, CSS
- **Bundler:** Vite

## Getting Started
To run this project locally, clone the repository and install the dependencies:

```bash
git clone https://github.com/reinier-dev/Vector-W-B.git
cd Vector-W-B
npm install
npm run dev
```
